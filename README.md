# NSE Option Chain Analyzer

A Python-based NSE Option Chain Analyzer built using **Python**, **Pandas**, and **Plotly**.

The project analyzes the NSE Option Chain CSV and extracts useful market insights such as:

## Features

- Read NSE Option Chain CSV
- Automatic data cleaning
- Call & Put Open Interest analysis
- PCR (Put Call Ratio)
- ATM Strike estimation
- Support & Resistance detection
- Interactive Plotly charts

## Technologies

- Python
- Pandas
- NumPy
- Plotly
- Google Colab
